# EchoMateLite

A lightweight social media platform built using Flask and MySQL.

## Features

- User Registration & Login
- Create Posts
- Follow / Unfollow Users
- Like Posts
- Notifications
- User Search
- Direct Messaging
- Admin Dashboard
- Responsive Mobile UI

## Tech Stack

- Python Flask
- MySQL
- Bootstrap 5
- Docker

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

## Docker

```bash
docker build -t echomatelite .
docker run -p 8000:8000 echomatelite
```
