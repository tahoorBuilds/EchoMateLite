# EchoMateLite 🚀

EchoMateLite is a social media web application built with Flask and MySQL. Users can create accounts, share posts, upload profile pictures, like posts, comment, follow other users, send messages, and receive notifications.

## Features

* User Registration & Login
* Profile Management
* Upload Profile Pictures
* Create, Edit & Delete Posts
* Like & Unlike Posts
* Comment System
* Follow & Unfollow Users
* User Search
* Post Search
* Inbox & Messaging
* Notifications
* Admin Dashboard
* Responsive Bootstrap UI

## Local Setup

```bash
git clone <repository-url>
cd EchoMateLite

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python app.py
```
